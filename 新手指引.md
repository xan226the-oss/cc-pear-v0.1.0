# cc-pear 新手指引

## 1. 下载

打开下载页：

https://github.com/xan226the-oss/cc-pear-v0.1.0/releases/tag/v0.1.0

下载 `cc-pear-v0.1.0.dmg`。

如果你点的是直接下载链接，浏览器会直接下载文件，不会显示 GitHub 页面说明，这是正常的。

## 2. 安装

1. 打开 `cc-pear-v0.1.0.dmg`。
2. 把 `cc-pear.app` 拖到 `Applications` 文件夹。
3. 打开「应用程序」，找到 `cc-pear`。

## 3. 第一次打开被 macOS 拦截怎么办

因为 cc-pear 目前没有 Apple Developer 公证，macOS 可能会提示：

`Apple 无法验证 cc-pear 是否包含恶意软件`

这样处理：

1. 点「完成」，不要点「移到废纸篓」。
2. 打开「系统设置 > 隐私与安全性」。
3. 滚到页面底部，找到 cc-pear 的安全提示。
4. 点「仍要打开」或「允许打开」。
5. 回到「应用程序」，再次打开 `cc-pear`。

以后再次打开就不会这样拦截。

## 4. 第一次启动要做什么

第一次打开 cc-pear，会弹出欢迎窗口。

你可以设置截图快捷键，也可以点：

`稍后设置，先使用 Control + 2`

默认截图快捷键是：

`Control + 2`

## 5. cc-pear 可以做什么

- 自动记录复制过的文字、链接、图片和文件。
- 小窗默认显示最近一天内容，也可以切换到七天或收藏。
- 点星标可以收藏某条记录，收藏内容永久保留。
- 可以选择多条内容，按顺序确认后，到目标位置按 `Command + V` 粘贴。
- 可以用截图快捷键截图，截图会进入 cc-pear。

## 6. 需要打开哪些权限

### 辅助功能

路径：

`系统设置 > 隐私与安全性 > 辅助功能`

用途：

用于按顺序自动粘贴多条内容。

如果你不使用“多条顺序粘贴”，可以先不开。

### 录屏与系统录音

路径：

`系统设置 > 隐私与安全性 > 录屏与系统录音`

用途：

用于截图。如果第一次截图时系统弹出权限提示，请允许 cc-pear。

## 7. 数据保存在哪里

cc-pear 只在本机保存数据，不上传。

数据位置：

`~/Library/Application Support/cc-pear/`

普通历史超过 7 天会自动删除；收藏内容不会自动删除。

## 8. 常用操作

- 打开小窗：点 Dock 里的 cc-pear，或点菜单栏里的 cc-pear。
- 截图：按你设置的截图快捷键，默认 `Control + 2`。
- 收藏：点记录右上角的星标。
- 删除：点记录右上角的垃圾桶。
- 复制单条：点记录右上角的复制按钮。
- 设置：点小窗顶部齿轮，或菜单栏里的「设置」。

## 9. 卸载

把「应用程序」里的 `cc-pear.app` 移到废纸篓即可。

如果想清空数据，请先在 cc-pear 的「设置」里点「清空全部数据」。
