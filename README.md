# cc-pear

cc-pear 是一个 macOS 剪贴板与截图小工具。它会在本机保存最近复制过的文字、链接、图片和文件；普通历史保留 7 天，收藏内容永久保留。

所有数据只保存在你的电脑本地，不会上传。

## 下载

下载 `cc-pear-v0.1.0.dmg`。

## 新手指引

第一次安装建议先看：

[cc-pear 新手指引](./新手指引.md)

里面写了下载、安装、macOS 安全拦截、首次启动、权限说明和常用操作。

## 安装

1. 打开 `cc-pear-v0.1.0.dmg`。
2. 把 `cc-pear.app` 拖到 `Applications` 文件夹。
3. 从「应用程序」或 Spotlight 打开 `cc-pear`。

## 第一次打开

因为 cc-pear 目前没有 Apple Developer 公证，macOS 可能会提示无法直接打开。

可以这样打开：

1. 在「应用程序」里找到 `cc-pear`。
2. 右键点击，选择「打开」。
3. 再点一次「打开」确认。

如果你看到“Apple 无法验证 cc-pear 是否包含恶意软件”的弹窗：

1. 点「完成」，不要点「移到废纸篓」。
2. 打开「系统设置 > 隐私与安全性」。
3. 滚到页面底部，找到 cc-pear 的安全提示。
4. 点「仍要打开」或「允许打开」。
5. 再回到「应用程序」里打开 cc-pear。

第一次启动时，cc-pear 会弹出新手指引：

1. 点按钮打开「辅助功能」权限，用于多条内容按顺序自动粘贴。
2. 点按钮打开「录屏与系统录音」权限，用于截图。
3. 小窗快捷键默认是 `Control + 1`，截图快捷键默认是 `Command + 2`。截图快捷键也可以首次启动时自定义。

## 使用

- Dock 图标：打开 cc-pear 小窗。
- 小窗快捷键：默认 `Control + 1`，可以打开或收起小窗。
- 菜单栏入口：打开小窗、截图、设置、退出。
- 菜单栏「新手指引」：随时重新查看权限和快捷键说明。
- 截图快捷键：默认 `Command + 2`，可以在设置里修改。
- 顺序粘贴：按顺序点卡片选择，按 `Enter` 确认，再到目标位置按 `Command + V`。
- 星标：收藏某条记录，收藏后永久保留。
- 设置：可以修改小窗快捷键、截图快捷键、开启开机自启、打开数据文件夹、清空全部数据。

## 数据保存

数据位置：

`~/Library/Application Support/cc-pear/`

普通历史超过 7 天会自动从后台记录里删除，包括文字、链接、富文本、图片和文件记录；未收藏的历史图片文件也会从本地 Media 文件夹清理。收藏不会自动删除。

## 系统要求

macOS 14 或更高版本。
