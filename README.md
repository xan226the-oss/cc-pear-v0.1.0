# cc-pear

cc-pear 是一个 macOS 剪贴板与截图小工具。它会在本机保存最近复制过的文字、链接、图片和文件；普通历史保留 7 天，收藏内容永久保留。

所有数据只保存在你的电脑本地，不会上传。

## 下载

下载 `cc-pear-v0.1.0.dmg`。

## 安装

1. 打开 `cc-pear-v0.1.0.dmg`。
2. 把 `cc-pear.app` 拖到 `Applications` 文件夹。
3. 从「应用程序」或 Spotlight 打开 `cc-pear`。

## 第一次打开

因为 cc-pear 目前没有 Apple Developer 公证，macOS 可能会提示无法直接打开。

可以这样打开：

1. 在「应用程序」里找到 `cc-pear`。
2. 右键点击，选择「打开」。
3. 再点一次「打开」确认。

第一次启动时，cc-pear 会让你设置截图快捷键。你也可以选择「稍后设置，先使用 Control + 2」。

## 使用

- Dock 图标：打开 cc-pear 小窗。
- 菜单栏入口：打开小窗、截图、设置、退出。
- 截图快捷键：默认 `Control + 2`，可以在设置里修改。
- 星标：收藏某条记录，收藏后永久保留。
- 设置：可以修改截图快捷键、开启开机自启、打开数据文件夹、清空全部数据。

## 数据保存

数据位置：

`~/Library/Application Support/cc-pear/`

普通历史超过 7 天会自动删除；收藏不会自动删除。

## 系统要求

macOS 14 或更高版本。
